Frequently Asked Questions (FAQs)

How do I add a new item?

Navigate to the Manage Items page, fill out the details in the "Add Item" form, and ensure the Item ID is unique.

How can I edit an existing item?

Go to the Manage Items page. You must enter the exact Item Name in the Update section to modify its details.

Will my data be saved after I close the browser?

No. As per the system design, data is only stored temporarily during your active session. It will be cleared once the browser is closed.

Troubleshooting Guide

  • Error: "Item ID already exists": Each item must have a unique identifier. Try using a different ID format (e.g., ITEM002).
  • Item not updating: Ensure you typed the Item Name exactly as it appears in the database. The system uses the name as the key for updates.
  • Prices/Quantities not accepting input: The system strictly validates numeric fields. Make sure you are not entering letters or special characters.
